# Overnight Mutual Funds — Risk Analysis & Crash Prediction

Exploratory Data Analysis and ML-based crash detection across 22 Indian 
overnight mutual funds using 55,000+ historical NAV records.

## What's inside
- EDA across 4 stress events: 2008 Crisis, Taper Tantrum, IL&FS 2018, COVID-19 2020
- ML classification pipeline: Logistic Regression, Random Forest, Gradient Boosting
- 11 engineered features: rolling volatility, return momentum, seasonal signals
- Confusion matrix, Recall, Precision, False Alarm Rate analysis

## Results
- Best model: Logistic Regression — 72%+ crash detection recall
- Analyzed 8,382 high-risk events across all funds

## Tech Stack
Python · Pandas · Scikit-learn · Matplotlib · Seaborn

## How to run
1. Open `Group11_Final.ipynb` in Jupyter
2. Ensure data file is in the same directory
3. Run all cells top to bottom
